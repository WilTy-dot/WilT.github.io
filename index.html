Predicting Social Media Engagement for Science Communication Posts using Machine Learning and Text Analytics

The aim of this project was to better understand what makes a science communication post on social media successful in terms of user engagement, particularly for science communication posts promoted through influencer marketing on Instagram. In this research, data from 10 influencers on Instagram were scraped, collected and analyzed to obtain key features in the post which make the post successful with a high engagement rate. Primarily, textual analysis was carried out and four categories of features (Post metadata, Text, LIWC, Sentiment) were obtained. Machine learning models were trained and tested on this data to obtain a prediction accuracy of 74.50% on the classification problem of predicting the science communication postâ€™s success on Instagram

Data Cleaning in SQL

In this project, I cleaned a housing data set using SQL Server. I specifically standardized the date format and the categorical variable " SoldAsVacant" using respectively the CONVERT and the CASE statement methods, then I populated the missing addresses using the ISNULL function. I also broken down the addresses to extract cities and states data using various methods that include SUBSTRING/CHARINDEX and PARSENAME/REPLACE. I then removed duplicates using ROW_NUMBER, CTE and the windows function PARTITION BY. I ended the cleaning by deleting few unused columns
