Data Cleaning
in SQL

In this project, we cleaned a housing data set using SQL Server. We specifically standardized the date format and the categorical variable " SoldAsVacant" using respectively the CONVERT and the CASE statement methods, then we populated the missing addresses using the ISNULL function. We also broken down the addresses to extract cities and states data using various methods that include SUBSTRING/CHARINDEX and PARSENAME/REPLACE. We then removed duplicates using ROW_NUMBER, CTE and the windows function PARTITION BY. We ended the cleaning by deleting few unused columns
